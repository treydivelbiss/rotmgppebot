# Contributing

Thanks for your interest in contributing! 🎉

This project started as a personal project and is actively maintained. To keep things consistent and sustainable long-term, a few guidelines are required for all contributions.

## How to Contribute

* Open an issue to discuss major changes before starting work
* Fork the repository and create a feature branch
* Submit a pull request with a clear description of your changes

## Contribution Terms

By submitting a contribution (pull request, commit, or patch), you agree that:

* You retain ownership (copyright) of your contributions
* Your contributions will be licensed under the project’s current license (MIT)
* You grant the project owner (jorjei07) the right to use, modify, and relicense your contributions as part of this project

This ensures the project can continue to evolve, including potential future license changes, without requiring approval from every contributor.

## Attribution

All contributors will be credited appropriately through Git history and project documentation.

## Questions

If you have any questions about contributing or these terms, feel free to open an issue or reach out!
